# 世界日報新聞魚 -- 角色設定

## 身分

**新聞魚**是一位新聞主播，不是吉祥物。它以讀遍所有通訊社的權威感來播報新聞。魚的造型純粹是視覺元素——播報的聲音從不提及魚的身分。

## 個性

應做：
- 像見多識廣的業內人士
- 自信、權威、直接
- 用淺顯易懂的語言解釋複雜主題
- 以明確的利害關係傳達事實

不應做：
- 使用魚類雙關語或水下用語
- 使用卡通幽默、迷因語言或海底詞彙
- 承認自己是一條魚
- 扮演「搞笑吉祥物」的角色

## 視覺構造

魚以 Remotion 分層合成渲染，帶有音訊驅動的口型同步。

### 口型系統

- **幀數**: 6 個口型狀態 (`mouth_0.png` 到 `mouth_5.png`)
- **驅動**: 來自 `@remotion/media-utils` `visualizeAudio` 的峰值音訊振幅
- **頻率範圍**: 取樣 1--40（低頻語音範圍）
- **平滑化**: 5 幀移動平均
- **靜音閾值**: 0.005
- **敏感度**: 0.04
- **資源位置**: `pipeline/video_templates/fish_lipsync/public/mouth/`

## 背景

海綿寶寶風格的水下場景，由 Gemini 生成。

- **比例**: 2:3（配合直式畫面）
- **風格**: 水彩紋理、藍綠漸層、卡通花朵、漂浮氣泡、手繪感
- **限制**: 無角色、無文字、無物件——僅背景圖案
- **變體**: 7 個 (`bg_original.png`, `bg_00.png` 到 `bg_05.png`)

## 聲音

- **供應商**: ElevenLabs TTS
- **格式**: MP3
- **模型**: `eleven_multilingual_v2`（支援華語）
- **聲音**: Stacy (`hkfHEbBvdQFNY4uWHqRF`) -- 台灣華語女聲
- **語言**: 繁體中文（Traditional Mandarin）
- **特色**: 專業、清晰、對話式的播報——符合頻道手冊中定義的「業內人士簡報」語氣

## 字幕

直接燒入影片畫面的字幕。

| 屬性       | 值                                                |
|-----------|---------------------------------------------------|
| 字體家族   | Inter, Avenir Next, Helvetica Neue, Arial, sans-serif |
| 字體大小   | 66px                                              |
| 字體粗細   | 900 (black)                                       |
| 行高       | 1.04                                              |
| 字距       | 0.01em                                            |
| 寬度       | 88% 畫面寬度                                      |
| 對齊       | 置中                                              |

字幕提示使用 ElevenLabs 強制對齊時間戳搭配 jieba 中文分詞計時。每個提示定義一個 `startSec`/`endSec` 視窗，在此期間文字可見。中文字幕每次顯示2-4個詞。

## 開頭

- **時間軸**: TTS 語音在影片開始時立即播放
- **視覺**: 整個場景從第 0 幀開始可見——魚、桌面、背景、媒體

## 結尾

- **音訊**: `breaking_news_outro.wav` 在 TTS 語音結束後播放
- **音量**: 1.0x
- **時間軸**: 在 `outroStartSeconds` 開始
